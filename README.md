Civic Issue Management System (CIMS)

Project Overview
The Civic Issue Management System is a web-based application developed to help citizens report and track civic-related issues such as road damage, water leakage, electricity faults, and sanitation problems. The system enables administrators to manage complaints efficiently and ensure timely resolution, improving communication between citizens and civic authorities.

Objectives
• Provide a simple and user-friendly platform for reporting civic issues
• Enable authorities to monitor, track, and resolve complaints
• Maintain transparency in the complaint resolution process
• Improve civic service management and accountability

Technologies Used
Backend: Python (Flask)
Frontend: HTML, CSS, JavaScript
Database: SQLite / MySQL
Version Control: Git and GitHub

Features
• User registration and login system
• Issue reporting with category and detailed description
• Admin panel for managing reported issues
• Issue status tracking (Pending, In Progress, Resolved)
• Search and filter functionality for complaints
• Complaint history management

How to Run the Project

Clone the repository
git clone https://github.com/sarthaknegi07-sn/civic-issue-management-system.git

Navigate to the project directory
cd civic-issue-management-system

Install required dependencies
pip install -r requirements.txt

Run the application
python app.py

Open a web browser and visit
http://127.0.0.1:5000/
